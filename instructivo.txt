Al clonar un repositorio react, se debe ejecutar "npm install" para instalar ciertas dependencias
y luego ejecutar "npm run dev" o "npm start" <--(frontend) <-- vista principal

Para django ejecutamos "pip install -r requirements.txt" que instalara los requerimientos extra para el correcto
funcionamiento de nuestro backend.<--(backend) <--(base de datos)
